# B30 STM32 测试例程
STM32 Library for B30 Sensor

<br/>

## 测试配件

**开发板:** SeaRobotix STM32 Test Board

<br/>

## 通信协议

IIC （逻辑电平：2.5-3.6 V）

<br/>
	
## 线序定义

线序|颜色|定义  
:-:|:-:|:-:
PC11|白|SDA 数据线   
PC10|绿|SCL 时钟线
VCC|红|2.5-5.5 V  
GND|黑|地
